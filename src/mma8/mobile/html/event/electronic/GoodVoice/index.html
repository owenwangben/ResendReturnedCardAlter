活動名稱： 永豐好聲音 交易聽歌趣！

活動期間： 2015/7/1-8/31

活動對象： 永豐行動銀行會員

活動說明： 於活動各階段交易期間內，登入永豐行動銀行APP並完成以下任務，就有機會獲得演唱會雙人門票乙組，共計40組(80張票)，一個ID僅限一次中獎機會

．任務1：各階段交易期間使用語音功能說「永豐好聲音」並完成永豐行動銀行登入，可獲得抽獎機會(一個身分證字號限一次抽獎機會)

．任務2：各階段交易期間登入永豐行動銀行APP並完成金融交易(當次交易手續費大於0元)，完成交易次數越多抽獎機會就越多!

  • (1)於本行MMA金融交易網首頁及行動銀行的「最新消息」公告得獎名單。
  • (2)於2015/7/28、8/12、8/26、9/15公告活動各階段得獎名單
  • (3)獎通知與獎品寄送：將以客戶於寄送贈品時間前15天留存於永豐MMA金融交易網之通訊資料為準，如有異動請於期限前更新完成，以確保得獎通知及奬品得正確寄送。如何修改個人基本資料

  • 四、對於透過任何不當程式進行大量封包發送、任何意圖破壞此次活動之不正常網路傳輸行為，或利用不當方式影響本活動公平性之人，主辦單位得取消其參加資格的權利，並就其所造成之損害，請求損害賠償。
  • 五、客戶參與本活動時，即同意接受本活動之活動辦法與注意事項規範，並接受本活動結果，如有違反，主辦單位得取消其參加或領獎資格。
  • 六、7/22、8/7、8/21及9/8由永豐銀行見證人公開抽獎，並於各抽獎日隔日起3日內以E-mail及手機簡訊通知得獎人。(請得獎者自行更新網站上所留下的E-mail、手機及地址，以確保收到通知，逾期未領者，視同自動放棄)。
  • 七、主辦單位保留擴大、修改、取消、終止該活動、更改活動截止日期及更換贈品的權利，如有未盡事宜，以活動網站公告為主。如有任何爭議，主辦單位保留最終決議權。
