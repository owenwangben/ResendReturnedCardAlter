如何使用基金速配？
  • Q : 使用基金速配客戶需符合什麼資格？
    A : (1)本行DBU客戶且須已申請網路銀行交易功能(約轉或非約轉)。
    (2)符合主動推介資格之客戶。
  • Q : 基金速配適用什麼商品？
    A : 目前只適用基金。
  • Q : 基金速配包含哪些交易類型？
    A : 包含基金申購（單筆及RSP）、轉換、贖回及贖回再申購；不含效率投資法。
  • Q : 手續費前收及後收型基金，都適用基金速配嗎？
    A : 前收型及後收型基金都適用。但部份類型有交易限制，例如：保本型及額控基金不適用。
  • Q : 客戶收到交易確認資料，多久需完成確認？
    A : 交易保留一天。有效確認期間為理專交易送出後的次一營業日pm3:30 前，客戶待確認頁提供每筆交易有效日／時間的提醒。
  • Q : 基金速配交易，客戶確認時間有何限制嗎？
    A : 客戶於營業時間內確認為當日交易，非營業時間確認為預約交易。「後收型基金」、「贖回再申購」及「全部贖回不續扣」限營業時間確認。「國內貨幣市場型基金」AM10:30前確認為當日申購交易。
