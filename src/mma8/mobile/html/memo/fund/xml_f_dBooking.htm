注意事項
  1. 交易時間
    週一至週五09：00至15：30，超過當日交易時間，則視為次一營業日之交易。
  2. 執行「取消預約交易」必須於交易生效日(營業日)當日09:00前取消該筆交易，若超過生效日當日(營業日) 09:00後，此筆交易已生效，無法取消。若欲取消已生效交易，請於15:30前，攜帶身分證明文件親洽臨櫃辦理。例如：預約8/20生效之交易，則必須於8/20 09:00以前取消該筆預約資料，若超過8/20 09:00以後，則此筆交易已生效，無法取消該筆交易。
  3. 如欲刪除「新申購但尚未執行扣款之定時定額基金交易」，須於定時定額扣款日「前一營業日」21：00前刪除該筆交易。（上述21：00將依本行系統實際處理時間而定，可能會有些許時間落差，故建議您儘早處理您的交易，避免影響個人權益）。例如：新申購但尚未執行扣款之定時定額基金交易預定於11/26(一)扣款，若欲刪除此筆交易需於 11/23(五)21：00前刪除，若逾時則無法刪除該筆交易。
  4. 您辦理「預約交易」，請務必在「生效日」之前一營業日確認約定扣款帳戶內有足夠之金額可供扣款（含申購及轉換手續費），以免發生餘額不足，導致該筆預約交易失敗。
  5. 有關基金應負擔之費用（含分銷費用）已揭露於基金公開說明書及投資人須知，投資人可至基金資訊觀測站（http://www.fundclear.com.tw）下載，或至基金公司網站查閱。
