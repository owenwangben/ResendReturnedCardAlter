• 臺幣定存注意事項
    1. 交易時間
      1. 台灣地區營業時間週一至週五9：00~23：00。其他時間交易為次一營業日計息之定存交易。
    2. 定存注意事項
      1. 定存期別：最短為1個月，最長為36個月。
      2. 定存最低金額為新台幣1萬元；零存整付最低扣款金額乘以期間須大於等於新台幣1萬元。
      3. 定存續存期限最長為20年。例如 : 若您辦理的定存為12個月，則最多只能再續存19次。
      4. 下列定存開單利率僅適用「永豐銀行台幣大額存款牌告利率」：
        1. 單筆定期存款超過1,000萬元（含）以上者；
        2. 單筆定期儲蓄存款超過1,000萬元（含）以上者；
        3. 單筆定期儲蓄存款超過1億元（含）以上者。
      5. 定存之到期日若遇假日則會順延至下一營業日入帳。
      6. 若辦理自動續存，自動續存時以續存當時牌告利率為續存利率。
    3. 預約交易
      1. 預約交易係指營業時間23:00後至次一營業日9:00前之交易，預約交易之定存起息點為次一營業日。
      2. 請您務必於預約交易成功後檢視您的帳戶實際存款餘額是否足夠，以免發生交易執行時因餘額不足，以致預約扣款失敗，本行將不另行通知。
      3. 當預約交易一旦確認送出後，請務必至「查詢/取消預約定存」功能，查詢您預約定存交易狀態是否已成功委託受理。
      4. 由於預約交易成功受理不代表您次一營業日之定存交易已成功交易，故請務必於次營業日上午九點後自行至「查詢帳戶/明細」及「查詢定存明細」查詢是否成功交易，如有交易失敗之情形，本行將不另行通知。
    4. 其他
      1. 本項交易僅提供『綜合存款帳戶』客戶辦理。
      2. 透過網路交易所辦理之定存不核發定存單，您可以至任一分行補登您的存摺，即可顯示您的存單帳號。
      3. 法人戶承作定期性存款，適用一般存款牌告利率之定存累積承作金額不得超過新台幣1,000萬（含）元；超過者且仍需開單之法人戶，請洽營業單位辦理。以大額存款牌告利率承作之存單交易則不在此限。
      4. 本項交易本行尚未開放外籍人士辦理。
    5. 定期存款中途解約利息之計算
      1. 未存滿一個月時，不予計息。
      2. 存滿一個月以上中途解約者，照實際存款期間足月部份之本行已掛牌前一較低期別定期存款利率八折計算。
      3. 前述各款牌告利率，以存入當日之牌告利率為準，但採用牌告利率機動計息之存款，按實際存款期間，如遇本行牌告利率調整，應同時改按新牌告利率分段計息。
      4. 以上各款調整後之利率不得低於本行活期存款牌告利率。

  • 外幣定存注意事項

      1. 台灣地區營業時間週一至週五9：00~23：00。非營業時間，暫停交易。
