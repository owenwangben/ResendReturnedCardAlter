注意事項
  1. 交易時間：
    1. 台灣地區營業時間週一至週五09：00~15：30。
  2. 金額限制：
    1. 涉及新臺幣結匯，每筆交易金額或每日累計最高限額(結購、結售分開計算)不得達等值新臺幣50萬元；如結匯幣別為人民幣，依「銀行業辦理外匯業務管理辦法」規定，每人每日交易限額為人民幣2萬元。前述交易限額與所有臨櫃及電子化業務通路合併計算。
    2. 單一營業日原幣匯款單筆及累計金額不得超過(含)等值新台幣50萬元。人民幣原幣匯出時，每人每日累計不得逾人民幣8萬元。
    3. 國外匯款之原幣匯出，每筆交易金額不得逾等值新臺幣200萬元。
    4. 大陸匯款之原幣匯出：
      1. 若為人民幣，僅限本國個人辦理，每人每日單筆或累計均不得逾人民幣8萬元。
      2. 其他幣別，每筆交易金額不得逾等值新臺幣200萬元。
  3. 費用說明：
    1. 匯出匯款手續費為每筆新臺幣100元。
    2. 郵電費為每筆新臺幣400元。
    3. 全額匯達則另外加收全額匯達手續費每筆新台幣750元。
    4. 上述費用將連同匯款金額一併從帳戶中扣取，若「扣款帳號」為外幣帳戶，則該費用將折合等值外幣後扣款。
  4. 匯款費用負擔說明：
    1. 若您於「匯款費用負擔」欄位選擇「SHA」，則每筆匯款費用為新臺幣500元(含匯出匯款手續費及郵電費)。除本行收取費用外，國外中間銀行亦會自匯款金額中收取相關費用。
    2. 若您於「匯款費用負擔」欄位選擇「OUR」，則此筆匯款金額將全額匯達受款銀行。本行除收取匯款費用每筆新臺幣500元外，將另外再收取全額匯達手續費新臺幣750元，合計匯款費用為新臺幣1,250元。
  5. 其他交易須知：
    1. 外籍人士需持有效之居留證始得辦理匯出匯款。若持有效期限一年以內之居留證，則僅限原幣匯出。
    2. 本服務未開放法人戶及外籍人士身份辦理人民幣匯款交易。
    3. 所有受款人帳號均需事先辦理約定。
    4. 若為大陸匯款且幣別為人民幣時，請您務必提供大陸銀行行號(CNAPS CODE)以及受款人電話號碼。
    5. 當您於大陸匯款選擇一日通時，若受款銀行非為一日通參加行或交易時間已逾中午12點，則本行將改採一般匯款流程匯出。
    6. 煩請務必至【查詢國外/大陸匯款】查詢交易進度，以確認交易是否申請成功。
    7. 若因資料有誤或不全與政府法令更改以致匯款無法匯出時，本行將由客服專員與您聯絡，故請您於「聯絡電話」欄位填入白天可聯繫您本人的電話。若當日15:30前無法聯繫到您本人，以致無法完成款項匯出時，本交易視同未成立。
