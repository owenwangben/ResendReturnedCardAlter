• Q : 請問什麼是房貸彈性繳款，有何特色呢？
    A : (1)房貸彈性繳款是讓您每月還款金額更有彈性，可約定最低只繳息（無條件進位至百元），最高為原本息攤還月付金之3倍（無條件捨去至百元）；而且單次申請最長可達24期的彈性還款方案。
    (2)線上申辦快速便利。讓您可以靈活運用房貸資金，調整財務需求。
  • Q : 請問申請房貸彈性繳款有申請次數的限制嗎？
    A : 只要符合申請資格條件即可申請，並無申請次數限制。
  • Q : 請問我剛申請過房貸彈性繳款，可以馬上再申請嗎?最多可以申請幾次呢？
    A : 只要符合申請資格條件即可申請，並無申請頻率及次數之限制。
  • Q : 請問公司戶可以申請房貸彈性繳款嗎？
    A : 申請房貸彈性繳款僅限本行符合申請資格條件之個人房貸戶。
  • Q : 我在網路申請房貸彈性繳款需要提供任何文件或資料嗎？
    A : 您只要符合申請資格即可進入網站申請，並確認約定內容即可，毋需提供任何文件或資料。
  • Q : 房貸彈性繳款申請成功後可以取消或變更嗎？
    A : 您可直接登入於查詢終止頁面申請取消即可。如果符合申請資格可以直接登入申請頁面直接申請修改變更。
  • Q : 申請房貸彈性繳款需要支付手續費嗎？
    A : (1)網路申請房貸彈性繳款每一筆分號收取手續費新台幣100元。
    (2)臨櫃申請房貸彈性繳款每次新台幣3,000~10,000元。
