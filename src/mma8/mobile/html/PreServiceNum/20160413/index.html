如何使用預約服務？
  • Q : 我可以透過什麼方式進行預約服務？
    A : 您可以使用永豐行動銀行APP(iOS/Android)，預約指定分行和指定時段，永豐銀行將為您專屬服務。
  • Q : 使用預約服務需為網路銀行會員嗎?
    A : 不需要，此項服務不用登入網銀也沒有網銀會員的限制，只要下載永豐行動銀行APP(iOS/Android)，進入「分行/ATM」即可使用預約服務。
  • Q : 預約服務可以預約非當日嗎?
    A : 可以的，預約服務可以預約3個營業日內的時段，請您多加利用，以利彈性安排您個人的時間。
  • Q : 使用預約服務，我拿到預約編號後，去分行該怎麼使用?
    A : 請您務必於預約時間內前往您預約的分行，於取號機上選擇「預約服務」項目，輸入預約編號，取得預約號碼牌後，請您稍候預約櫃台叫號。
  • Q : 我想預約的時段額滿了，怎麼辦?
    A : (1)若預約時段已額滿無法預約，請您預約其他時段或親臨分行現場辦理業務。
    (2)若您於現場取號並等候叫號並無名額之限制。
  • Q : 如果我當天遲到了或走錯分行，怎麼辦?
    A : 請您重新預約或直接於分行行內取號機選擇您想辦理的業務項目，取得現場號碼牌後，等候分行人員叫號為您服務。(如有預先填單資料，也請您同步提供預約編號和原預約分行名稱給分行同仁，以利節省您辦理業務的時間)
  • Q : 如果我早到了，怎麼辦?
    A : 請您當天至預約分行於取號機選擇「預約服務」項目，輸入預約編號。因非您指定預約時段，故取得現場號碼牌後並等候櫃台叫號。(如有預先填單資料，也請您同步提供預約編號給分行同仁，節省您辦理業務的時間)
  • Q : 為什麼我不能使用預約服務了?
    A : 為維持系統之安全性，每一行動裝置每日預約次數上限為3次，若超過上限將無法預約，欲使用預約服務，請隔日再試。
  • Q : 我可以在線上先行填寫哪些表單?
    A : 預約服務目前提供您臺幣現金存款、臺幣現金提款、本行帳戶轉帳、臺幣匯款的交易表單，請您先行線上填寫，但您仍需臨櫃完成交易，所需提供之憑證或文件同臨櫃交易辦理之方式。
  • Q : 預先填單可以自己帶入我的帳號嗎? 我不想填寫這麼多數字?
    A : 可以的，請您先行登入永豐行動銀行APP(iOS/Android)，使用預約服務之預先填單功能，線上填寫表單時，可以選擇本人帳號，不用另行填寫帳號。
  • Q : 我的預約紀錄不見了?
    A : 當您透過行動裝置完成預約服務時，本行將會留存預約紀錄(僅限原裝置)，預約紀錄保存期間為3個月，超逾3個月本行將會清除該筆紀錄，但若您刪除永豐行動銀行APP(iOS/Android)，原本的預約紀錄有可能會被清除，不便之處，尚請見諒。
