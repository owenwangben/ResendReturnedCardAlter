信用卡掛失

以下方法為模擬 Ajax 取得資料回來後 render 在 lightbox 裡的方式，請自行串接使用。

用法1:
openlbox('#err');
$('#err h3').text(標題);
$('#err .desc p').text(描述);

用法2:
openlbox('#cs-msg');
$('#cs-msg h3').text(標題);
$('#cs-msg .desc p').text(描述);

HTML:
<a href="#" class="open-lbox" data-lboxid="#err">操作流程錯誤(驗證token序號不符)</a>
<a href="#" class="open-lbox" data-lboxid="#cs-msg">訊息(掛失多張卡片)</a>

以上方法不想用的話或者直接參考 html 原始檔內的 script 自行斟酌使用也行。
